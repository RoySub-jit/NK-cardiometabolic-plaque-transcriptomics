## GSE198339 reproducibility note

The verified external PBMC analysis used the official processed GSE198339 dataset comprising 9,368 PBMCs from eight male participants living with HIV, four with and four without atherosclerosis. Participant-level non-HDL cholesterol was calculated as total cholesterol minus HDL cholesterol from the accompanying clinical metadata.

The corrected analytical audit, participant-level outputs, and verified statistics are provided in `../reproducibility_audit/GSE198339/` and `../results_final_main/GSE198339_verified/`.

Raw public dataset files are not redistributed in this repository and should be obtained directly from their public source repositories.
